Formation OPENCLASSROOMS - Développeur Web / Projet N°2 - Transformer son C.V. en site Web

Second projet de la formation Développeur Web d'OPENCLASSROOMS.

Transformer son C.V. en site Web en respectant la charte graphique imposée par OPENCLASSROOMS (Voir le fichier image "EXEMPLE C.V. OC").


OBJECTIFS DU PROJET:
    - Transformation du C.V. en site Web ;
    - Création d'un lien vers le repository Github ou Gitlab où repose le code du projet ;
    - Versionner le code sur Github ou Gitlab ;
    - Rendre le repository accessible  ;
    - Créer un lien vers la page web hébergée en ligne sur Github Pages ou Gitlab Pages, pour présenter le C.V. 
  

CONTRAINTES DU PROJET:
  
    - Usage de Font Awesome (Icônes personnalisables) & Material Design (Palettes de couleurs) ;
    - Site Web accessible pour toutes plateformes (Ordinateur & appareils mobiles), format "Responsive" ;
    - Usage des langages HTML & CSS uniquement et sans Framework ;
    - Le code ne doit contenir aucune erreur ni alerte au validateur HTML & CSS.
  
  
OUTILS & SUPPORTS USITÉS:

    - GitHub / Dépôts et versionnage du code ;
    - W3C Validation Service / Validateur du code HTML & CSS.
